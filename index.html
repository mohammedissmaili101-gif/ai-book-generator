🚀 How to Use

1

Get Free Groq API Key

Go to console.groq.com → Create API Key → Copy it (starts with gsk_)

2

Add API Key to GitHub Secrets

Repository → Settings → Secrets → New Secret → Name: GROQ_API_KEY

3

Enable GitHub Pages

Settings → Pages → Source: Deploy from branch → gh-pages

4

Run the Workflow

Actions → "Generate Professional Ebook" → Run workflow → Enter book title → Choose language

5

Get Your Book!

Wait ~2 minutes → Your book appears here on GitHub Pages → Share on TikTok! 🎉
